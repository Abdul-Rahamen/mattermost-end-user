# تشغيل الأوامر المائلة

استخدام أمر مائل سهل مثل كتابة رسالة. بدلاً من نص الرسالة، تبدأ الأوامر المائلة بحرف slash: `/`.

على سبيل المثال، إذا كنت تريد تسجيل الخروج من منصة تعاون باستخدام لوحة المفاتيح فقط، يمكنك إدخال `/logout` في صندوق نص الرسالة واختيار **إرسال**.

بما أن رسالتك تبدأ بـ `/`، تعرف منصة تعاون أنها أمر مائل، وتقوم بالإجراء المحدد لهذا الأمر.

## إنشاء أوامر مائلة مخصصة

هل تبحث عن إنشاء أوامر مائلة جديدة مخصصة لنسخة تعاون الخاصة بك؟ تفقد [توثيق المطور](https://developers.mattermost.com/integrate/slash-commands/custom/) للتعرف على المزيد حول إنشاء أوامر مائلة مخصصة لمسارات عملك.
