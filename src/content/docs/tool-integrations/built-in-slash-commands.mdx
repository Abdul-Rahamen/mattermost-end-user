# الأوامر المائلة المدمجة

فيما يلي الأوامر المائلة المدمجة المتوفرة في [مساحة عملك](/access-your-workspace/access-your-workspace/). 

:::tip

هل تبحث عن المزيد من الأوامر المائلة؟ راجع [توثيق المطور للأوامر المائلة المخصصة](https://developers.workspace.com/integrate/slash-commands/custom/) للحصول على تفاصيل حول إنشاء أوامر مخصصة.

:::

## دعوة الأشخاص

- دعوة شخص واحد باستخدام `/invite user1` أو `/invite @user1`.
- دعوة مجموعة مستخدمين مخصصة باستخدام `/invite @usergroup`.
- دعوة عدة أشخاص باستخدام `/invite @user1 @user2`.
- دعوة شخص واحد إلى قناة معينة باستخدام `/invite @user1 ~channel1`، أو `/invite @user1 channel1`.
- دعوة عدة أشخاص إلى قنوات متعددة باستخدام `/invite @user1 @user2 ~channel1 ~channel2`.
- دعوة الأشخاص عبر البريد الإلكتروني باستخدام `/invite_people {name@domain.com, ...}`.

## الانضمام إلى قنوات ومغادرتها وكتم صوتها

- الانضمام إلى قناة معينة باستخدام `/join {channel-name}` أو `/open {channel-name}`.
- مغادرة قناة باستخدام `/leave`.
- كتم صوت قناة باستخدام `/mute` أو `/mute {channel-name}` لإيقاف تشغيل إشعارات سطح المكتب، والبريد الإلكتروني، والإشعارات الفورية للقناة الحالية أو المحددة.
- إزالة شخص ما من قناة باستخدام `/kick {@username}` أو `/remove {@username}`.

## بدء أو الانضمام إلى محادثة

- بدء محادثة في قناة أو موضوع باستخدام `/call start`.
- الانضمام إلى محادثة في قناة أو موضوع باستخدام `/call join`.

## إدارة المحادثات

- إرسال رسالة مباشرة إلى شخص ما باستخدام `/msg {@username} {message}`، أو إرسال رسالة جماعية إلى عدة أشخاص باستخدام `/groupmsg {@username1, @username2, @username3,...} {message}`.
- عرض النص ككتلة كود باستخدام `/code {text}`.
- طي معاينات الصور تلقائيًا باستخدام `/collapse`، وإظهارها تلقائيًا باستخدام `/expand`.
- صدّر النص إلى نفسك باستخدام `/echo {message} {delay in seconds}` أو `/me {message}`.
- الرد بإيماءة استسلام باستخدام `/shrug {message}`.
- البحث في نص الرسائل باستخدام `search {text}`.

## تعيين تواجدك وحالتك

- تعيين تواجدك باستخدام `/away`، `/offline`، `/online`، أو `/dnd`.
- تعيين حالة مخصصة باستخدام `/status {emoji_name} {descriptive status_message}`، مثل `/status sick Feeling unwell and taking time off to recover`. امسح حالتك الحالية باستخدام `/status clear`.

## إدارة القنوات

- تعديل عنوان القناة باستخدام `/header {text}` أو الغرض من القناة باستخدام `/purpose {text}`.
- إعادة تسمية قناة باستخدام `/rename {text}`.

## المزيد من الأوامر المائلة المفيدة

- فتح توثيق المنتج لتعاون باستخدام `/help`.
- فتح سوق الإضافات داخل المنتج باستخدام `/marketplace`.
- عرض قائمة اختصارات لوحة المفاتيح باستخدام `/shortcuts`.
- فتح شاشة **الإعدادات** باستخدام `/settings`.
- تسجيل الخروج من منصة تعاون باستخدام `/logout`.
