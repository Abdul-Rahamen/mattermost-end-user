الأوامر المائلة المدمجة التالية متاحة في مساحة عملك في منصة تعاون.

:::tip
تبحث عن أوامر مائلة مخصصة؟ راجع وثائق المطورين الخاصة بـ [الأوامر المائلة المخصصة](https://developers.mattermost.com/integrate/slash-commands/custom/) للحصول على تفاصيل حول إنشاء أوامر مخصصة.
:::

### دعوة الأشخاص

- دعوة شخص واحد باستخدام `/invite user1` أو `/invite @user1`.
- دعوة مجموعة مستخدمين مخصصة باستخدام `/invite @usergroup`.
- دعوة عدة أشخاص باستخدام `/invite @user1 @user2`.
- دعوة شخص واحد إلى قناة محددة باستخدام `/invite @user1 ~channel1`، أو `/invite @user1 channel1`.
- دعوة عدة أشخاص إلى عدة قنوات باستخدام `/invite @user1 @user2 ~channel1 ~channel2`.
- دعوة الأشخاص عبر البريد الإلكتروني باستخدام `/invite_people {name@domain.com, ...}`.

### الانضمام إلى القنوات أو مغادرتها أو كتمتها

- الانضمام إلى قناة محددة باستخدام `/join {channel-name}` أو `/open {channel-name}`.
- مغادرة قناة باستخدام `/leave`.
- كتم قناة باستخدام `/mute` أو `/mute {channel-name}` لإيقاف إشعارات سطح المكتب، والبريد الإلكتروني، والإشعارات المنقولة للقناة الحالية أو المحددة.
- إزالة شخص من قناة باستخدام `/kick {@username}` أو `/remove {@username}`.

### بدء أو الانضمام إلى مكالمة

- بدء مكالمة في قناة أو سلسلة محادثات باستخدام `/call start`.
- الانضمام إلى مكالمة في قناة أو سلسلة محادثات باستخدام `/call join`.

### إدارة المحادثات

- إرسال رسالة خاصة لشخص باستخدام `/msg {@username} {message}`، أو إرسال رسالة جماعية إلى عدة أشخاص باستخدام `/groupmsg {@username1, @username2, @username3,...} {message}`.
- عرض النص ككتلة برمجية باستخدام `/code {text}`.
- طي معاينات الصور تلقائياً باستخدام `/collapse`، وتوسيعها تلقائياً باستخدام `/expand`.
- إعادة نص الرسالة إليك باستخدام `/echo {message} {delay in seconds}` أو `/me {message}`.
- الرد بتعبير مستعجب باستخدام `/shrug {message}`.
- البحث في نص الرسائل باستخدام `search {text}`.

### تعيين التوافر والحالة

- تعيين [توافرك](/end-user-guide/preferences/set-your-status-availability:set-your-availability) باستخدام `/away`، `/offline`، `/online`، أو `/dnd`.
- تعيين [حالة مخصصة](/end-user-guide/preferences/set-your-status-availability:set-a-custom-status) باستخدام `/status {emoji_name} {descriptive status_message}`، مثل `/status sick Feeling unwell and taking time off to recover`. امسح حالتك الحالية باستخدام `/status clear`.

### إدارة القنوات

- تعديل ترويسة القناة باستخدام `/header {text}` أو الغرض من القناة باستخدام `/purpose {text}`.
- إعادة تسمية قناة باستخدام `/rename {text}`.

### أوامر مفيدة إضافية

- فتح توثيق منتج منصة تعاون باستخدام `/help`.
- فتح متجر السوق في المنتج باستخدام `/marketplace`.
- عرض قائمة اختصارات لوحة المفاتيح باستخدام `/shortcuts`.
- فتح شاشة **الإعدادات** باستخدام `/settings`.
- تسجيل الخروج من منصة تعاون باستخدام `/logout`.
