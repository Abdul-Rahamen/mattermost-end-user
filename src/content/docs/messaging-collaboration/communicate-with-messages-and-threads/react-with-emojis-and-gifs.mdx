## إدارة وتخصيص الرموز

### اختيار لون البشرة الافتراضي
يمكنك تحديد لون البشرة الذي سيظهر افتراضياً للرموز البشرية.

<Steps>
1. افتح منتقي الرموز التعبيرية.
2. اختر أيقونة **لون البشرة** في الزاوية العلوية.
3. حدد الدرجة المفضلة لديك.
</Steps>

<Image 
  src={emojiSkinTone} 
  alt="تحديد لون البشرة" 
  style={{ width: '100%', maxWidth: '400px', borderRadius: '8px', display: 'block', margin: '20px auto' }} 
/>

### تحميل رموز مخصصة
يمكنك إضافة رموز فريدة لمساحة العمل الخاصة بك (ما لم يمنع مسؤول النظام ذلك).

<Steps>
1. من منتقي الرموز، اختر **الرموز المخصصة **.
   <Image src={addCustomEmojiBtn} alt="زر الإضافة" style={{ width: '100%', maxWidth: '500px', borderRadius: '8px', display: 'block', margin: '10px auto' }} />

2. أدخل اسماً فريداً للرمز (سيعمل مع الإكمال التلقائي).
3. اختر **تحديد (Select)** وارفع الصورة (يفضل أن تكون مربعة، بصيغة JPG/PNG/GIF، وبحجم أقل من 512 KiB).
4. اضغط **حفظ**.
   <Image src={addCustomEmojiModal} alt="نافذة الرفع" style={{ width: '100%', maxWidth: '500px', borderRadius: '8px', display: 'block', margin: '10px auto' }} />
</Steps>

#### حذف الرموز المخصصة
يمكنك حذف الرموز التي قمت بتحميلها عبر الدخول إلى قسم **الرموز المخصصة** داخل المنتقي، والضغط على **حذف (Delete)** بجانب الرمز المطلوب.
