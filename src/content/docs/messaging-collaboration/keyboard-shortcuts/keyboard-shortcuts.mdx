تساعدك اختصارات لوحة المفاتيح في منصة تعاون على تحقيق أقصى استفادة من لوحة المفاتيح لزيادة كفاءة عملك، سواء كنت تستخدم منصة تعاون عبر متصفح الويب أو تطبيق سطح المكتب.

:::tip[نصائح إضافية]

- تتوفر اختصارات إضافية؛ يمكنك الاطلاع على توثيق [اختصارات لوحة مفاتيح الفريق](/messaging-collaboration/organize-using-teams/team-keyboard-shortcuts) وتفاصيل [تنسيق النصوص](/messaging-collaboration/communicate-with-messages-and-threads/format-messages#text-style).
- لعرض قائمة الاختصارات المتاحة داخل منصة تعاون، اضغط على `Ctrl` + `/` في ويندوز ولينكس، أو `⌘` + `/` في نظام macOS، أو استخدم الأمر المائل `/shortcuts`.

![مراجعة قائمة اختصارات لوحة المفاتيح المتاحة باستخدام الأمر المائل /shortcuts](../../../../images/keyboard-shortcuts.png)
:::

## التنقل بين القنوات

تدعم المتصفحات المتوافقة وتطبيق منصة تعاون لسطح المكتب الاختصارات التالية للتنقل بين القنوات:

| الوصف                                                                                                       | على Mac                | على Windows و Linux      |
| :---------------------------------------------------------------------------------------------------------- | :--------------------- | :----------------------- |
| القناة أو الرسالة الخاصة السابقة في الشريط الجانبي.                                                         | <kbd>⌥ + ↑</kbd> | <kbd>Alt + ↑</kbd> |
| القناة أو الرسالة الخاصة التالية في الشريط الجانبي.                                                         | <kbd>⌥ + ↓</kbd> | <kbd>Alt + ↓</kbd> |
| القناة أو الرسالة الخاصة السابقة التي تحتوي على رسائل غير مقروءة.                                           | <kbd>⌥ + ⇧ + ↑</kbd> | <kbd>Alt + Shift + ↑</kbd> |
| القناة أو الرسالة الخاصة التالية التي تحتوي على رسائل غير مقروءة.                                           | <kbd>⌥ + ⇧ + ↓</kbd> | <kbd>Alt + Shift + ↓</kbd> |
| تحديد آخر منشور في القناة كـ "غير مقروء".                                                                   | <kbd>⌥ + النقر على القناة</kbd> | <kbd>Alt + النقر على القناة</kbd> |
| - إذا لم يتم تحديد نص: فتح نافذة **البحث عن القنوات**.<br/>- إذا تم تحديد نص: إنشاء رابط تشعبي. | <kbd>⌘ + K</kbd> | <kbd>Ctrl + K</kbd> |
| فتح نافذة **الرسائل الخاصة**.                                                             | <kbd>⌘ + ⇧ + K</kbd> | <kbd>Ctrl + Shift + K</kbd> |
| فتح نافذة **الإعدادات**.                                                                         | <kbd>⌘ + ⇧ + A</kbd> | <kbd>Ctrl + Shift + A</kbd> |
| فتح "الإشارات" الأخيرة.                                                                          | <kbd>⌘ + ⇧ + M</kbd> | <kbd>Ctrl + Shift + M</kbd> |
| نقل التركيز إلى حقل إدخال الرسائل الرئيسي في القناة.                                                        | <kbd>⌘ + ⇧ + L</kbd> | <kbd>Ctrl + Shift + L</kbd> |
| فتح أو إغلاق الشريط الجانبي الأيمن.                                                                         | <kbd>⌘ + .</kbd> | <kbd>Ctrl + .</kbd> |
| توسيع أو تصغير عرض الشريط الجانبي الأيمن.                                                                   | <kbd>⌘ + ⇧ + .</kbd> | <kbd>Ctrl + Shift + .</kbd> |
| نقل التركيز إلى حقل **البحث** والبحث في القناة الحالية.                                                     | <kbd>⌘ + ⇧ + F</kbd> | <kbd>Ctrl + Shift + F</kbd> |
| البحث عن القنوات غير المقروءة أو البحث في كافة القنوات.                                                     | <kbd>⌘ + ⇧ + U</kbd> | <kbd>Ctrl + Shift + U</kbd> |
| فتح أو إغلاق تفاصيل **معلومات القناة**.                                                                  | <kbd>⌘ + ⇧ + I</kbd> | <kbd>Ctrl + Shift + I</kbd> |

## رفع الملفات

يمكنك استخدام الاختصار التالي لرفع الملفات في كافة الأنظمة:

| الوصف    | على Mac   | على Windows و Linux |
| :------- | :-------- | :------------------ |
| رفع ملف. | <kbd>⌘ + U</kbd> | <kbd>Ctrl + U</kbd> |

## إدارة الرسائل

تدعم المتصفحات وتطبيق سطح المكتب الاختصارات التالية للتعامل مع الرسائل:

| الوصف                                                                  | على Mac                | على Windows و Linux    |
| :--------------------------------------------------------------------- | :--------------------- | :--------------------- |
| إعادة كتابة الرسالة السابقة أو الأمر المائل الذي أدخلته (في حقل فارغ). | <kbd>⌘ + ↑</kbd> | <kbd>Ctrl + ↑</kbd> |
| إعادة كتابة الرسالة التالية أو الأمر المائل الذي أدخلته (في حقل فارغ). | <kbd>⌘ + ↓</kbd> | <kbd>Ctrl + ↓</kbd> |
| الرد على أحدث رسالة في القناة الحالية (في حقل فارغ).                   | <kbd>⇧ + ↑</kbd> | <kbd>Shift + ↑</kbd> |
| تعديل آخر رسالة أرسلتها في القناة الحالية (في حقل فارغ).               | <kbd>↑</kbd> | <kbd>↑</kbd> |
| الإكمال التلقائي لاسم المستخدم .                            | <kbd>@[character] + Tab</kbd> | <kbd>@[character] + Tab</kbd> |
| الإكمال التلقائي لاسم القناة (~).                                      | <kbd>~[character] + Tab</kbd> | <kbd>~[character] + Tab</kbd> |

## تنسيق الرسائل

استخدم هذه الاختصارات لتنسيق المحتوى بسرعة:

| الوصف                     | على Mac         | على Windows و Linux  |
| :------------------------ | :-------------- | :------------------- |
| جعل النص عريضاً.   | <kbd>⌘ + B</kbd> | <kbd>Ctrl + B</kbd> |
| جعل النص مائلاً. | <kbd>⌘ + I</kbd> | <kbd>Ctrl + I</kbd> |
| تنسيق النص كرابط.         | <kbd>⌘ + ⌥ + K</kbd> | <kbd>Ctrl + Alt + K</kbd> |

## التنقل وتسهيل الوصول

تساعد هذه الاختصارات على تحسين إمكانية الوصول  وسرعة التنقل:

| الوصف                                         | على Mac       | على Windows و Linux |
| :-------------------------------------------- | :------------ | :------------------ |
| الانتقال إلى القناة السابقة في سجل تصفحك.     | <kbd>⌘ + [</kbd> | <kbd>Alt + ←</kbd> |
| الانتقال إلى القناة التالية في سجل تصفحك.     | <kbd>⌘ + ]</kbd> | <kbd>Alt + →</kbd> |
| تظليل النص حتى السطر السابق (في حقل الإدخال). | <kbd>⇧ + ↑</kbd> | <kbd>Shift + ↑</kbd> |
| تظليل النص حتى السطر التالي (في حقل الإدخال). | <kbd>⇧ + ↓</kbd> | <kbd>Shift + ↓</kbd> |
| إنشاء سطر جديد.                               | <kbd>⇧ + Enter</kbd> | <kbd>Shift + Enter</kbd> |

:::note
على الرغم من دعم منصة تعاون للغات وتخطيطات لوحة المفاتيح القياسية، قد لا تعمل بعض الاختصارات إذا كنت تستخدم برامج تخصيص مفاتيح تتعارض مع اختصارات المتصفح الافتراضية.
:::

## المكالمات

| :---------------------------------------------- | :--------------------------- | :------------------------------------ |
| بدء مكالمة أو الانضمام إليها في القناة الحالية. | <kbd>⌘ + ⌥ + S</kbd> | <kbd>Ctrl + Alt + S</kbd> |
| كتم الصوت أو إلغاء الكتم.                       | <kbd>⌘ + ⇧ + Space</kbd> | <kbd>Ctrl + Shift + Space</kbd> |
| رفع اليد أو خفضها.                              | <kbd>⌘ + ⇧ + Y</kbd> | <kbd>Ctrl + Shift + Y</kbd> |
| مشاركة الشاشة أو إيقاف مشاركتها.                | <kbd>⌘ + ⇧ + E</kbd> | <kbd>Ctrl + Shift + E</kbd> |
| إظهار أو إخفاء قائمة المشاركين.                 | <kbd>⌥ + P</kbd> أو <kbd>⌘ + ⇧ + P</kbd> | <kbd>Alt + P</kbd> أو <kbd>Ctrl + Shift + P</kbd> |
| مغادرة المكالمة الحالية.                        | <kbd>⌘ + ⇧ + L</kbd> | <kbd>Ctrl + Shift + L</kbd> |
| الضغط المستمر للتحدث.            | <kbd>Space</kbd> | <kbd>Space</kbd> |
