تساعدك اختصارات لوحة المفاتيح في منصة تعاون على تحقيق أقصى استفادة من لوحة المفاتيح لزيادة كفاءة عملك، سواء كنت تستخدم منصة تعاون عبر متصفح الويب أو تطبيق سطح المكتب.

:::tip[نصائح إضافية]

- تتوفر اختصارات إضافية؛ يمكنك الاطلاع على توثيق [اختصارات لوحة مفاتيح الفريق](/messaging-collaboration/organize-using-teams/team-keyboard-shortcuts) وتفاصيل [تنسيق النصوص](/messaging-collaboration/communicate-with-messages-and-threads/format-messages#text-style).
- لعرض قائمة الاختصارات المتاحة داخل منصة تعاون، اضغط على `Ctrl` + `/` في ويندوز ولينكس، أو `⌘` + `/` في نظام macOS، أو استخدم الأمر المائل `/shortcuts`.

![مراجعة قائمة اختصارات لوحة المفاتيح المتاحة باستخدام الأمر المائل /shortcuts](../../../../images/keyboard-shortcuts.png)
:::

## التنقل بين القنوات

تدعم المتصفحات المتوافقة وتطبيق منصة تعاون لسطح المكتب الاختصارات التالية للتنقل بين القنوات:

| على Windows و Linux | على Mac | الوصف |
| :--- | :--- | :--- |
| <kbd>Alt + ↑</kbd> | <kbd>⌥ + ↑</kbd> | القناة أو الرسالة الخاصة السابقة في الشريط الجانبي. |
| <kbd>Alt + ↓</kbd> | <kbd>⌥ + ↓</kbd> | القناة أو الرسالة الخاصة التالية في الشريط الجانبي. |
| <kbd>Alt + Shift + ↑</kbd> | <kbd>⌥ + ⇧ + ↑</kbd> | القناة أو الرسالة الخاصة السابقة التي تحتوي على رسائل غير مقروءة. |
| <kbd>Alt + Shift + ↓</kbd> | <kbd>⌥ + ⇧ + ↓</kbd> | القناة أو الرسالة الخاصة التالية التي تحتوي على رسائل غير مقروءة. |
| <kbd>Alt + النقر على القناة</kbd> | <kbd>⌥ + النقر على القناة</kbd> | تحديد آخر منشور في القناة كـ "غير مقروء". |
| <kbd>Ctrl + K</kbd> | <kbd>⌘ + K</kbd> | - إذا لم يتم تحديد نص: فتح نافذة **البحث عن القنوات**.<br/>- إذا تم تحديد نص: إنشاء رابط تشعبي. |
| <kbd>Ctrl + Shift + K</kbd> | <kbd>⌘ + ⇧ + K</kbd> | فتح نافذة **الرسائل الخاصة**. |
| <kbd>Ctrl + Shift + A</kbd> | <kbd>⌘ + ⇧ + A</kbd> | فتح نافذة **الإعدادات**. |
| <kbd>Ctrl + Shift + M</kbd> | <kbd>⌘ + ⇧ + M</kbd> | فتح "الإشارات" الأخيرة. |
| <kbd>Ctrl + Shift + L</kbd> | <kbd>⌘ + ⇧ + L</kbd> | نقل التركيز إلى حقل إدخال الرسائل الرئيسي في القناة. |
| <kbd>Ctrl + .</kbd> | <kbd>⌘ + .</kbd> | فتح أو إغلاق الشريط الجانبي الأيمن. |
| <kbd>Ctrl + Shift + .</kbd> | <kbd>⌘ + ⇧ + .</kbd> | توسيع أو تصغير عرض الشريط الجانبي الأيمن. |
| <kbd>Ctrl + Shift + F</kbd> | <kbd>⌘ + ⇧ + F</kbd> | نقل التركيز إلى حقل **البحث** والبحث في القناة الحالية. |
| <kbd>Ctrl + Shift + U</kbd> | <kbd>⌘ + ⇧ + U</kbd> | البحث عن القنوات غير المقروءة أو البحث في كافة القنوات. |
| <kbd>Ctrl + Shift + I</kbd> | <kbd>⌘ + ⇧ + I</kbd> | فتح أو إغلاق تفاصيل **معلومات القناة**. |

## رفع الملفات

يمكنك استخدام الاختصار التالي لرفع الملفات في كافة الأنظمة:

| على Windows و Linux | على Mac | الوصف |
| :--- | :--- | :--- |
| <kbd>Ctrl + U</kbd> | <kbd>⌘ + U</kbd> | رفع ملف. |

## إدارة الرسائل

تدعم المتصفحات وتطبيق سطح المكتب الاختصارات التالية للتعامل مع الرسائل:

| على Windows و Linux | على Mac | الوصف |
| :--- | :--- | :--- |
| <kbd>Ctrl + ↑</kbd> | <kbd>⌘ + ↑</kbd> | إعادة كتابة الرسالة السابقة أو الأمر المائل الذي أدخلته (في حقل فارغ). |
| <kbd>Ctrl + ↓</kbd> | <kbd>⌘ + ↓</kbd> | إعادة كتابة الرسالة التالية أو الأمر المائل الذي أدخلته (في حقل فارغ). |
| <kbd>Shift + ↑</kbd> | <kbd>⇧ + ↑</kbd> | الرد على أحدث رسالة في القناة الحالية (في حقل فارغ). |
| <kbd>↑</kbd> | <kbd>↑</kbd> | تعديل آخر رسالة أرسلتها في القناة الحالية (في حقل فارغ). |
| <kbd>@[character] + Tab</kbd> | <kbd>@[character] + Tab</kbd> | الإكمال التلقائي لاسم المستخدم . |
| <kbd>~[character] + Tab</kbd> | <kbd>~[character] + Tab</kbd> | الإكمال التلقائي لاسم القناة (~). |

## تنسيق الرسائل

استخدم هذه الاختصارات لتنسيق المحتوى بسرعة:

| على Windows و Linux | على Mac | الوصف |
| :--- | :--- | :--- |
| <kbd>Ctrl + B</kbd> | <kbd>⌘ + B</kbd> | جعل النص عريضاً. |
| <kbd>Ctrl + I</kbd> | <kbd>⌘ + I</kbd> | جعل النص مائلاً. |
| <kbd>Ctrl + Alt + K</kbd> | <kbd>⌘ + ⌥ + K</kbd> | تنسيق النص كرابط. |

## التنقل وتسهيل الوصول

تساعد هذه الاختصارات على تحسين إمكانية الوصول  وسرعة التنقل:

| على Windows و Linux | على Mac | الوصف |
| :--- | :--- | :--- |
| <kbd>Alt + ←</kbd> | <kbd>⌘ + [</kbd> | الانتقال إلى القناة السابقة في سجل تصفحك. |
| <kbd>Alt + →</kbd> | <kbd>⌘ + ]</kbd> | الانتقال إلى القناة التالية في سجل تصفحك. |
| <kbd>Shift + ↑</kbd> | <kbd>⇧ + ↑</kbd> | تظليل النص حتى السطر السابق (في حقل الإدخال). |
| <kbd>Shift + ↓</kbd> | <kbd>⇧ + ↓</kbd> | تظليل النص حتى السطر التالي (في حقل الإدخال). |
| <kbd>Shift + Enter</kbd> | <kbd>⇧ + Enter</kbd> | إنشاء سطر جديد. |

:::note
على الرغم من دعم منصة تعاون للغات وتخطيطات لوحة المفاتيح القياسية، قد لا تعمل بعض الاختصارات إذا كنت تستخدم برامج تخصيص مفاتيح تتعارض مع اختصارات المتصفح الافتراضية.
:::

## المكالمات

عند تفعيل ميزة [المكالمات](/messaging-collaboration/collaborate-within-channels/make-calls-in-taawon)، تتوفر الاختصارات التالية:

| على Windows و Linux | على Mac | الوصف |
| :--- | :--- | :--- |
| <kbd>Ctrl + Alt + S</kbd> | <kbd>⌘ + ⌥ + S</kbd> | بدء مكالمة أو الانضمام إليها في القناة الحالية. |
| <kbd>Ctrl + Shift + Space</kbd> | <kbd>⌘ + ⇧ + Space</kbd> | كتم الصوت أو إلغاء الكتم. |
| <kbd>Ctrl + Shift + Y</kbd> | <kbd>⌘ + ⇧ + Y</kbd> | رفع اليد أو خفضها. |
| <kbd>Ctrl + Shift + E</kbd> | <kbd>⌘ + ⇧ + E</kbd> | مشاركة الشاشة أو إيقاف مشاركتها. |
| <kbd>Alt + P</kbd> أو <kbd>Ctrl + Shift + P</kbd> | <kbd>⌥ + P</kbd> أو <kbd>⌘ + ⇧ + P</kbd> | إظهار أو إخفاء قائمة المشاركين. |
| <kbd>Ctrl + Shift + L</kbd> | <kbd>⌘ + ⇧ + L</kbd> | مغادرة المكالمة الحالية. |
| <kbd>Space</kbd> | <kbd>Space</kbd> | الضغط المستمر للتحدث. |
