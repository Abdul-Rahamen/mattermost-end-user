تساعدك اختصارات لوحة المفاتيح في **منصة تعاون** على تحقيق أقصى استفادة من نظام التشغيل لزيادة كفاءة عملك اليومي وسرعته، سواء كنت تستخدم المنصة عبر متصفح الويب أو من خلال تطبيق سطح المكتب المخصص.

:::tip[نصائح إضافية لزيادة الإنتاجية]
* **المزيد من الاختصارات**: تتوفر توليفات مفاتيح إضافية؛ يمكنك مراجعة دليل [اختصارات لوحة مفاتيح الفريق](/messaging-collaboration/organize-using-teams/team-keyboard-shortcuts) والتعرف على قواعد [تنسيق النصوص المتقدمة](/messaging-collaboration/communicate-with-messages-and-threads/format-messages#text-style).
* **استدعاء اللوحة الفورية**: لعرض لوحة مرجعية تفاعلية تسرد جميع الاختصارات المتاحة داخل مساحة عملك، اضغط على اختصار <kbd>Ctrl</kbd> + <kbd>/</kbd> على أنظمة Windows و Linux، أو اختصار <kbd>⌘</kbd> + <kbd>/</kbd> على أنظمة macOS، أو اكتب الأمر المائل التالي في حقل الدردشة وأرسله: `/shortcuts`.

## التنقل بين القنوات

تدعم المتصفحات المتوافقة وتطبيقات سطح المكتب الاختصارات التالية للانتقال الفوري والذكي بين قنوات غرف الدردشة:

| الوظيفة والإجراء | على أنظمة Windows و Linux | على أنظمة macOS |
| :--- | :--- | :--- |
| **الانتقال للقناة أو الرسالة المباشرة السابقة** | <kbd>Alt</kbd> + <kbd>↑</kbd> | <kbd>⌥</kbd> + <kbd>↑</kbd> |
| **الانتقال للقناة أو الرسالة المباشرة التالية** | <kbd>Alt</kbd> + <kbd>↓</kbd> | <kbd>⌥</kbd> + <kbd>↓</kbd> |
| **الانتقال للقناة السابقة المحتوية على رسائل غير مقروءة** | <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>↑</kbd> | <kbd>⌥</kbd> + <kbd>⇧</kbd> + <kbd>↑</kbd> |
| **الانتقال للقناة التالية المحتوية على رسائل غير مقروءة** | <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>↓</kbd> | <kbd>⌥</kbd> + <kbd>⇧</kbd> + <kbd>↓</kbd> |
| **تحديد آخر منشور في القناة كـ "غير مقروء"** | <kbd>Alt</kbd> + <kbd>النقر على اسم القناة</kbd> | <kbd>⌥</kbd> + <kbd>النقر على اسم القناة</kbd> |
| **البحث السريع عن القنوات** / **إدراج رابط تشعبي** | <kbd>Ctrl</kbd> + <kbd>K</kbd> | <kbd>⌘</kbd> + <kbd>K</kbd> <br />*(يفتح نافذة البحث عن القنوات، وإذا كنت مضللاً لنص يصيغه كرابط)* |
| **فتح نافذة الرسائل المباشرة** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>K</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>K</kbd> |
| **فتح لوحة الإعدادات الشخصية** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>A</kbd> |
| **استعراض سجل الإشارات  الأخيرة** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>M</kbd> |
| **نقل التركيز فوراً إلى حقل كتابة الرسائل الرئيسي** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>L</kbd> |
| **فتح أو إغلاق الشريط الجانبي الأيمن** | <kbd>Ctrl</kbd> + <kbd>.</kbd> | <kbd>⌘</kbd> + <kbd>.</kbd> |
| **توسيع (ملء الشاشة) أو تصغير عرض الشريط الأيمن** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>.</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>.</kbd> |
| **التركيز على شريط البحث العام وفلترة القناة الحالية** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>F</kbd> |
| **البحث الشامل في كافة القنوات غير المقروءة** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>U</kbd> |
| **فتح أو إغلاق نافذة معلومات وتفاصيل القناة** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>I</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>I</kbd> |

## إدارة وصياغة الرسائل

تحكم في حركة وسجل منشوراتك وصياغتها داخل حقول الإدخال عبر الاختصارات التالية:

| الوظيفة والإجراء | على أنظمة Windows و Linux | على أنظمة macOS |
| :--- | :--- | :--- |
| **استدعاء وتكرار رسالتك/أمرك المائل السابق** | <kbd>Ctrl</kbd> + <kbd>↑</kbd> | <kbd>⌘</kbd> + <kbd>↑</kbd> *(يُشترط أن يكون حقل الإدخال فارغاً)* |
| **استدعاء وتكرار رسالتك/أمرك المائل التالي** | <kbd>Ctrl</kbd> + <kbd>↓</kbd> | <kbd>⌘</kbd> + <kbd>↓</kbd> *(يُشترط أن يكون حقل الإدخال فارغاً)* |
| **فتح نافذة الرد على أحدث رسالة في القناة الحالية** | <kbd>Shift</kbd> + <kbd>↑</kbd> | <kbd>⇧</kbd> + <kbd>↑</kbd> *(يُشترط أن يكون حقل الإدخال فارغاً)* |
| **التعديل السريع  المباشر لآخر رسالة أرسلتها** | <kbd>↑</kbd> | <kbd>↑</kbd> *(اضغط سهم الصعود وحقل الكتابة فارغ تماماً)* |
| **الإكمال التلقائي الذكي لاسم مستخدم/موظف** | <kbd>@اسم_المستخدم</kbd> + <kbd>Tab</kbd> | <kbd>@اسم_المستخدم</kbd> + <kbd>Tab</kbd> |
| **الإكمال التلقائي الذكي لروابط وأسماء القنوات** | <kbd>~اسم_القناة</kbd> + <kbd>Tab</kbd> | <kbd>~اسم_القناة</kbd> + <kbd>Tab</kbd> |

## تنسيق النصوص السريع

نسّق الكلمات والعبارات المحددة داخل صندوق صياغة الرسائل فوراً دون الحاجة لكتابة رموز Markdown يدوياً:

| نمط التنسيق البصري | على أنظمة Windows و Linux | على أنظمة macOS |
| :--- | :--- | :--- |
| **تغليظ وتثخين النص المظلل** (**Bold**) | <kbd>Ctrl</kbd> + <kbd>B</kbd> | <kbd>⌘</kbd> + <kbd>B</kbd> |
| **إمالة الخط للنص المظلل** (*Italic*) | <kbd>Ctrl</kbd> + <kbd>I</kbd> | <kbd>⌘</kbd> + <kbd>I</kbd> |
| **تحويل الكلمة المظللة إلى رابط تشعبي** | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>K</kbd> | <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>K</kbd> |

## التنقل وتسهيل الوصول (Accessibility & History)

توليفات متقدمة لإدارة مسار المؤشر وسجل تصفح مساحات العمل:

| الإجراء والوظيفة | على أنظمة Windows و Linux | على أنظمة macOS |
| :--- | :--- | :--- |
| **الرجوع للخلف في سجل تصفح القنوات** | <kbd>Alt</kbd> + <kbd>←</kbd> | <kbd>⌘</kbd> + <kbd>[</kbd> |
| **التقدم للأمام في سجل تصفح القنوات** | <kbd>Alt</kbd> + <kbd>→</kbd> | <kbd>⌘</kbd> + <kbd>]</kbd> |
| **تظليل وتحديد النص صعوداً سطر بستر** | <kbd>Shift</kbd> + <kbd>↑</kbd> | <kbd>⇧</kbd> + <kbd>↑</kbd> *(داخل حقل الكتابة)* |
| **تظليل وتحديد النص نزولاً سطر بسطر** | <kbd>Shift</kbd> + <kbd>↓</kbd> | <kbd>⇧</kbd> + <kbd>↓</kbd> *(داخل حقل الكتابة)* |
| **إنشاء سطر جديد داخل نفس الرسالة** | <kbd>Shift</kbd> + <kbd>Enter</kbd> | <kbd>⇧</kbd> + <kbd>Enter</kbd> |

:::caution[تنبيه البرامج المتعارضة]
يرجى الانتباه إلى أنه على الرغم من البناء القياسي للاختصارات وتوافقها مع اللغات الدولية، إلا أن استخدام تطبيقات طرف ثالث لإعادة تعيين أزرار الفأرة ولوحة المفاتيح (Key-mappers) قد تسبب تداخلاً وتعارضاً يحظر استجابة الخادم لبعض هذه الأوامر.
:::

## اختصارات إدارة المكالمات والاجتماعات

عند قيام الإدارة بتفعيل ميزة [المكالمات الصوتية والمرئية](/messaging-collaboration/collaborate-within-channels/make-calls-in-workspace) داخل القنوات، يمكنك استخدام الأدوات التالية:

| الإجراء الصوتي والمرئي | على أنظمة Windows و Linux | على أنظمة macOS |
| :--- | :--- | :--- |
| **بدء مكالمة جديدة أو الانضمام للمكالمة الجارية** | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>S</kbd> | <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>S</kbd> |
| **كتم الميكروفون أو إلغاء الكتم (Mute/Unmute)** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>Space</kbd> |

| **عرض أو إخفاء لوحة أسماء المشاركين في الاجتماع** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> أو <kbd>Alt</kbd> + <kbd>P</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>P</kbd> أو <kbd>⌥</kbd> + <kbd>P</kbd> |
| **المغادرة الفورية وإنهاء الاشتراك في المكالمة** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>L</kbd> |
| **الضغط المؤقت للتحدث المباشر (Push-to-Talk)** | <kbd>مفتاح المسافة Space</kbd> | <kbd>مفتاح المسافة Space</kbd> <br />*(يفتح الميكروفون طوال فترة ضغطك فقط)* |

## التحكم في حجم ومقياس العرض

| مقياس درجة الزوم | على أنظمة Windows و Linux | على أنظمة macOS |

| **زيادة حجم الخط والواجهة تدريجياً (تكبير)** | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>+</kbd> | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>+</kbd> |
