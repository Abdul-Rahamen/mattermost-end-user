## إنشاء قوائم التحقق والمهام

يمكنك إنشاء قالب عمل متكامل يحتوي على أقسام رئيسية وتندرج تحتها مهام فرعية محددة.

<Image 
  src={channelTasksChecklistTemplate} 
  alt="واجهة قالب قوائم التحقق وإضافة المهام" 
  style={{ width: '100%', maxWidth: '450px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '20px auto' }} 
/>

### خطوات إعداد القائمة:
<Steps>
1. اضغط على **إضافة قسم** لتقسيم مشروعك إلى مراحل أساسية.
2. اضغط على **إضافة مهمة** لإنشاء بند عمل جديد تحت القسم المحدد.
3. أدخل **عنوان المهمة** ووصفاً اختيارياً يوضح المطلوب إنجازه.
</Steps>

## تخصيص خيارات المهمة

يوفر شريط الأدوات المصغر أسفل كل مهمة أربعة خيارات متقدمة لإدارتها بدقة:

### 1. المسند إليه (تعيين المسؤول)
اضغط على أيقونة المستخدم لفتح قائمة أعضاء القناة وتحديد المسؤول عن تنفيذ هذه المهمة. يمكنك البحث عن الاسم مباشرة باستخدام معرف المستخدم (`@username`).

<Image 
  src={channelATaskEmployeeAssignment} 
  alt="قائمة تعيين الموظف المسند إليه المهمة" 
  style={{ width: '100%', maxWidth: '400px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '20px auto' }} 
/>

### 2. تاريخ الاستحقاق
اضغط على أيقونة التقويم لتحديد الموعد النهائي للتنفيذ. يوفر النظام خيارات سريعة مثل (اليوم، غداً، الأسبوع القادم) أو تحديد تاريخ مخصص وساعة محددة.

<Image 
  src={channelATaskDurationExecution} 
  alt="نافذة تحديد تاريخ ووقت استحقاق المهمة" 
  style={{ width: '100%', maxWidth: '400px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '20px auto' }} 
/>

### 3. تشغيل الأوامر 
تسمح لك أيقونة الشرطة المائلة (`/`) بربط المهمة بأحد أوامر النظام التلقائية (مثل ضبط الحالة `away`، أو جلب أحداث التقويم `cal`، أو عرض كتل برمجية `code`) ليتم تنفيذ الأمر فور التفاعل مع المهمة.

<Image 
  src={channelATaskTargetCommand} 
  alt="قائمة ربط المهام بأوامر النظام التلقائية" 
  style={{ width: '100%', maxWidth: '400px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '20px auto' }} 
/>

### 4. إجراءات المهام (الأتمتة الذكية)
تتيح لك أيقونة البرق (`⚡`) أتمتة الأنشطة بناءً على محفزات معينة. على سبيل المثال، يمكنك ضبط النظام ليقوم بـ **وضع علامة بأن المهمة مكتملة** تلقائياً عند نشر رسالة تحتوي على كلمات مفتاحية محددة داخل القناة.
