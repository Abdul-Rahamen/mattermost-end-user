## إنشاء قوائم التحقق والمهام

يمكنك بناء وتصميم قالب عمل متكامل ومحكم يحتوي على أقسام هيكلية كبرى تندرج تحتها مهام تفصيلية فرعية.

<Image 
  src={channelTasksChecklistTemplate} 
  alt="واجهة صياغة قوالب قوائم التحقق وإضافة تقسيمات المهام" 
  style={{ width: '100%', maxWidth: '450px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '2rem auto' }} 
/>

### خطوات إعداد القائمة:
<Steps>
1. انقر فوق زر **إضافة قسم** لتقسيم مشروعك أو خطتك التشغيلية إلى مراحل أساسية مرتبة.
2. انقر فوق زر **إضافة مهمة** لإنشاء بند عمل تفصيلي جديد تحت القسم المستهدف.
3. أدخل **عنوان المهمة** وصغ وصفاً اختيارياً يشرح المعايير والمطلوب إنجازه بدقة.
</Steps>

## تخصيص معلمات وخيارات المهمة

يوفر شريط الأدوات المصغر الملحق بأسفل كل مهمة أربعة خيارات برمجية متقدمة لحوكمة وإدارة البند بدقة:

### 1. المسند إليه (تعيين المسؤول)
اضغط على أيقونة المستخدم لفتح قائمة أعضاء القناة وتحديد الشخص المسؤول عن التنفيذ. يمكنك الفلترة والبحث عن الاسم مباشرة باستخدام معرّف الحساب المسبوق برمز الإشارة (`@username`).

<Image 
  src={channelATaskEmployeeAssignment} 
  alt="قائمة فرز وتعيين الموظف المسؤول عن تنفيذ المهمة" 
  style={{ width: '100%', maxWidth: '400px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '2rem auto' }} 
/>

### 2. تاريخ ووقت الاستحقاق
اضغط على أيقونة التقويم لتحديد الموعد النهائي الصارم للتنفيذ. يوفر النظام خيارات جدولة سريعة (مثل: اليوم، غداً، الأسبوع القادم) أو اختيار تاريخ مخصص وساعة محددة بدقة عبر المقياس الزمني.

<Image 
  src={channelATaskDurationExecution}
  alt="نافذة تحديد الموعد النهائي وتاريخ استحقاق المهمة" 
  style={{ width: '100%', maxWidth: '400px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '2rem auto' }} 
/>

### 3. ربط وتشغيل الأوامر المائلة
تسمح لك أيقونة الشرطة المائلة (`/`) بربط المهمة بأحد أوامر النظام التلقائية المدمجة (مثل: ضبط الحالة `away`، جلب أحداث التقويم `cal`، أو استدعاء كتل برمجية `code`)؛ ليقوم الخادم بتنفيذ الأمر المائل برمجياً فور قيام العضو بنقر المهمة أو التفاعل معها.

<Image 
  src={channelATaskTargetCommand}

  style={{ width: '100%', maxWidth: '400px', borderRadius: '8px', border: '1px solid var(--sl-color-gray-5)', display: 'block', margin: '2rem auto' }} 
/>

### 4. إجراءات المهام وأتمتتها الذكية
تتيح لك أيقونة البرق التفاعلية (`⚡`) أتمتة الأنشطة التشغيلية بناءً على محفزات (Triggers) معينة. على سبيل المثال، يمكنك تهيئة النظام ليقوم بـ **وضع علامة بأن المهمة مكتملة** تلقائياً بمجرد قيام الزملاء بنشر أو إرسال رسالة تحتوي على كلمات مفتاحية معينة (Keywords) يتم رصدها برمجياً داخل القناة.
